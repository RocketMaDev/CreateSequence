# CreateSequence

一个Java小程序用来快速生成单双数序列

## 参数列表

|参数              |类型    |描述                   |
|-----------------|-------|----------------------|
|<-?/--help/无参数>|String |打印帮助                |
|start,end        |int,int|指定起止位置             |
|range            |int    |指定终止值，以1为起始值    |
|<-r/--reverse>   |String |决定在复制偶串时是否反转顺序|

## e.g.  
CreateSequence.jar 2,7 -r  
->  
3,5,7  
6,4,2
